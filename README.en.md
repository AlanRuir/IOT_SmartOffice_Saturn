# SmartOffice

#### Description
一款面向办公场景的办公助手，通过各种传感器对办公区电器进行控制，也可以对办公区温度、湿度等数据进行采集和监控,上位机采用微信小程序负责给用户提供用于调整、控制和查看办公区数据的客户端软件。
